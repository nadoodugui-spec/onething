1. 이 서비스가 하는 일

The One Thing은 "할 일을 모으는 도구"가 아니라 "가장 중요한 하나를 끝내게 하는 도구"입니다. 각자 매일 '오늘의 단 하나(원씽)'를 정해 화면의 주인공으로 올리고, 그걸 끝내기 전까지 나머지 할 일은 잠깁니다. 팀에서는 서로의 집중 상태가 보이고, 급하지 않은 요청은 집중이 끝날 때까지 조용히 기다렸다가 전달됩니다. 쪽지·업무 요청은 받은 사람의 할 일로 이어지고 진행·완료가 보낸 사람에게 자동으로 돌아오므로 "그거 어떻게 됐어요?"라는 확인 문화가 사라집니다.

2. 권한 조직도 — 누가 무엇을 할 수 있나

👑 최고관리자 (플랫폼 운영자 · 나두혁) 서비스 전체를 관리하는 유일한 최상위 권한
  • 플랫폼 관리 화면: 전체 회사·회원 수 확인, 이름/이메일 검색
  • 회원 계정(프로필) 삭제 — 유일하게 가능
  • 비밀번호 재설정 메일 발송(본인 해결이 안 될 때)
  • 보안 규칙·서버 설정 등 시스템 운영
▼
🛡 웹사이트 관리매니저 (예정) 최고관리자가 지정 — 플랫폼 운영을 돕는 중간 등급. 아직 도입 전(유료화 시점에 추가 예정)
▼   회사(워크스페이스)마다 아래 구조가 반복됩니다   ▼
🏢 오너 (회사를 만든 사람) 사내관리자의 모든 권한 + 직책 관리
  • 구성원을 사내관리자로 승격/해제 — 오너만 가능(화면·보안 규칙 모두)
  • 초대 코드 확인·재발급, 구성원 내보내기, 팀 통계, 관리 기록(감사 로그)
▼
💼 사내관리자 (오너가 승격한 구성원) 회사 안 사람 관리 — 단, 개인 계정은 못 건드림
  • 초대 코드 확인·재발급으로 새 구성원 초대
  • 구성원 내보내기(소속 해제 — 미완료 요청은 자동으로 넘겨받음, 6초 실행 취소 가능)
  • 팀 통계(통계의 '팀' 탭), 관리 기록 열람
  • 불가: 승격/해제(오너 전용), 구성원 프로필(이름·이메일) 수정, 비밀번호 관련, 계정 삭제
▼
👤 일반 구성원 회사에 소속된 회원
  • 개인 기능 전부 + 팀 기능 전부(쪽지·업무 요청·프로젝트·팀 현황)
  • 내 프로필(이름)·내 비밀번호는 본인만 관리
🙋 무소속 개인 회사에 참여하지 않은 회원
  • 개인 기능(원씽·할 일·List·타이머·통계)은 모두 사용
  • 팀 기능은 잠김 — 초대 코드로 참여하거나 회사를 만들면 열림

3. 시작하기

가입과 로그인

  • 사이트 접속 → 이메일+비밀번호 가입(8자 이상, 영문+숫자 포함) 또는 Google로 계속하기.
  • 가입하면 인증 메일이 발송됩니다. 메일함에서 확인해 주세요(미인증이어도 사용은 가능, 사용자 메뉴에 안내가 떠 있습니다).
  • 첫 로그인 시 팀에 보일 이름을 정합니다.
  • 비밀번호를 잊었다면 로그인 화면의 "비밀번호를 잊으셨나요?"로 본인이 직접 재설정 메일을 받습니다.

회사(워크스페이스) 만들기 / 참여하기

  • 👤 사용자 메뉴 → 워크스페이스 구획에서:
  • 만들기: 회사 이름을 입력하면 새 워크스페이스가 생기고 내가 오너가 됩니다. 6자리 초대 코드가 발급됩니다.
  • 참여: 받은 초대 코드 6자리를 입력하면 그 회사의 구성원이 됩니다.
  • 회사 없이도 개인 기능은 전부 쓸 수 있습니다.

4. 개인 사용법

오늘의 원씽 정하기

  • 왼쪽 페이지에 할 일을 적고, 그중 하나에서 ◉ 버튼을 누르면 그 일이 오른쪽 "THE ONE THING" 자리에 올라갑니다.
  • 원씽을 정하는 순간 내 상태가 자동으로 "원씽 중"이 되어 팀에 보입니다.
  • 원씽을 끝내기 전까지 나머지 할 일 목록은 잠깁니다(잠금 해제 방식은 설정에서 3초 길게 누르기/버튼 선택).

원씽 부수기(하위 단계)

  • 큰 일은 조각으로 나누세요. 원씽 카드 아래 "부수기"에 단계를 적으면 번호가 붙고, 지금 할 조각이 강조됩니다.

할 일 목록

  • 우선순위 3단계: Now(지금) · Next(다음) · Routine(루틴). 입력 후 Enter 조합키(Enter=Now 등 안내 표시)로 바로 지정.
  • 할 일마다 마감일·색·반복·하위 할일·검색 지원, 완료된 항목은 접혀서 정리됩니다.
  • +List 탭: 주제별 독립 목록(예: 아이디어, 장보기). 우클릭으로 이름 변경, 드래그로 순서 변경.
  • 휴지통(30일 보관), "나중에" 보관함, Ctrl+K 전체 검색.

집중 타이머

  • 원씽 카드의 타이머(기본 길이는 설정에서)를 켜면 상태가 "원씽 중 · ~N분 후 종료"로 표시됩니다(남은 시간 공개는 설정에서 끌 수 있음).
  • 시간이 끝나도 자동 완료되지 않고 알림만 옵니다. 5분 휴식 버튼 지원.

5. 팀 사용법

상태(프레즌스)

  • 네 가지: 원씽 중(보라 — 집중, 방해 금지) · 업무 중(초록) · 자리비움(회색) · 휴가 중(복귀일·대리인 표시).
  • 원씽을 정하면 자동으로 "원씽 중", 접속이 끊기면 자동 "자리비움".

쪽지와 업무 요청

  • 쪽지 = 짧은 메시지. 업무 요청 = 제목+서식 있는 본문(굵게·색·표·체크리스트)+파일 첨부+마감일+반복(매일/매주/매월).
  • 받는 사람이 원씽 중이면 알림이 바로 울리지 않고 조용히 쌓였다가, 집중이 끝나면 한꺼번에 전달됩니다.
  • 받은 요청은 ① 즉시완료(바로 끝냄) ② 내 할 일로 추가(Now/Next/Routine 선택) ③ 답장 ④ 전달 ⑤ 진행 상황 남기기로 처리합니다.
  • 할 일로 추가된 요청을 완료하면 보낸 사람에게 자동으로 완료가 통보됩니다.
  • 보낸 요청은 안읽음/읽음(시각)/완료(시각)가 표시되고, 재촉(리마인드)·마감 지연 표시를 지원합니다.
  • 상대가 읽기 전이라면 수정(상대에게 "(수정됨)+시각" 표시)과 회수(상대에게 "회수됨"으로 표시되고 내용은 숨김) 가능.
  • 자주 쓰는 요청은 템플릿으로 저장. 완료 90일이 지난 요청은 자동 정리됩니다.

팀 현황 사이드바

  • 툴바 왼쪽 팀 현황 버튼: 전 구성원의 상태·공개 원씽·집중 진행바가 보이고, 이름에 마우스를 올리면 쪽지/업무요청 버튼이 나옵니다.
  • 📌 고정하면 새로고침해도 열린 채 유지됩니다.

6. 프로젝트(칸반 보드)

  • 여러 명이 함께 하는 일은 프로젝트 탭으로: 할 일 / 진행 중 / 완료 3열 보드.
  • 카드에 담당자·마감일·설명·댓글. 드래그 또는 이동 버튼(휴대폰)으로 열 이동.
  • 카드의 "◉ 내 원씽으로"를 누르면 그 카드가 내 오늘의 원씽이 되고, 원씽을 완료하면 카드도 자동 완료됩니다.
  • 프로젝트는 초대제 — 멤버로 추가된 사람만 봅니다. 탭에 진행률이 표시됩니다.

7. 매일의 리듬

  • 오늘 브리핑: 하루의 시작에 마감 임박 요청·Now 할 일·내 프로젝트 카드 중에서 '오늘의 단 하나' 후보를 골라줍니다(건너뛰기 가능).
  • 도미노: 끝낸 원씽이 도미노처럼 쌓이는 시각화 — 매일 하나가 큰 성과가 되는 과정을 보여줍니다.
  • 통계: 연속 달성, 주간 요약, 12주 잔디(히트맵), 지켜진 집중 시간. 사내관리자·오너는 나/팀 탭으로 팀 현황도 봅니다.

8. 설정과 데이터

  • 개인: 테마(자동/라이트/다크), 집중 타이머 길이, 잠금 해제 방식(3초 길게 누르기/버튼).
  • 팀에 공개: 남은 시간 공개, 오늘의 원씽 공개(기본 꺼짐), 팀 활동 피드 표시.
  • 데이터: 백업 내보내기/불러오기(JSON 파일). 데이터는 클라우드에 실시간 동기화되어 PC·노트북·휴대폰 어디서나 이어집니다.
  • 한/영 전환 지원. 단축키: Ctrl+K 전체 검색, T 테마 전환 등.

9. 사내관리자 안내

  • 👤 사용자 메뉴에 구성원 관리 구획이 보입니다.
  • 초대: 초대 코드를 새 구성원에게 알려주면 됩니다. 코드가 유출되면 재발급으로 무효화하세요.
  • 내보내기: 소속만 해제됩니다(계정·개인 데이터는 그대로). 그 사람이 갖고 있던 미완료 요청은 나에게 자동 전달되고, 6초 안에 실행 취소할 수 있습니다.
  • 승격/해제(오너만): 구성원을 사내관리자로 올리거나 내립니다.
  • 모든 관리 행위는 관리 기록(감사 로그)에 남고 최근 5건이 표시됩니다.
  • 구성원이 비밀번호를 잊으면 → 로그인 화면에서 본인이 직접 재설정하도록 안내하세요. 그래도 안 되면 최고관리자에게 요청.

10. 최고관리자(플랫폼 운영) 안내

  • 운영자 계정으로 로그인하면 👤 사용자 메뉴 맨 아래 플랫폼 관리 → 전체 현황 보기가 나타납니다.
  • 상단 요약: 워크스페이스 수 · 전체 회원 수 · 무소속 회원 수.
  • 회사별 카드: 회사 이름, 구성원 수, 개설일, 초대 코드, 구성원 목록(이름·이메일·오너/관리자 표시·가입 시각).
  • 검색창에 회사 이름 또는 회원 이름/이메일을 입력하면 실시간으로 걸러집니다.
  • 각 회원 옆 버튼: 비밀번호 재설정(메일 발송) · 계정 삭제(프로필 말소 — 유일한 삭제 권한).

11. 보안 구조

  • 로그인: Google의 Firebase 인증(이메일+비밀번호 또는 Google 계정). 비밀번호는 서비스가 저장하지 않고 Google 인프라가 관리합니다.
  • 회사 간 격리: 회사의 데이터(요청·상태·프로젝트)는 그 회사 구성원만 읽고 쓸 수 있습니다 — 화면이 아니라 서버 규칙이 막으므로 우회 접근도 불가능합니다.
  • 개인 노트: 각자의 할 일·원씽 데이터는 본인 계정만 접근합니다.
  • 권한 최소화: 사내관리자는 소속·직책만, 직책은 오너만, 계정 삭제는 최고관리자만 — 2장 조직도 그대로 서버에서 강제됩니다.
  • 첨부파일은 파일당 1MB·요청당 2MB 제한.

12. 휴대폰에서 쓰기

  • 휴대폰 브라우저에서 사이트를 열고 "홈 화면에 추가"(아이폰: 공유 → 홈 화면에 추가)를 하면 앱처럼 아이콘이 생기고 전체 화면으로 실행됩니다.
  • 모바일 화면에 맞춰 자동으로 배치가 바뀌고, 칸반 카드는 이동 버튼으로 옮깁니다.
  • PC와 실시간 동기화 — 회사에서 정한 원씽을 이동 중에도 확인.

13. 자주 묻는 질문

  • Q. 회사를 옮기면 내 할 일은? — 개인 데이터는 계정에 붙어 있으므로 그대로 유지됩니다. 회사 데이터(요청 등)는 그 회사에 남습니다.
  • Q. 원씽 중인 사람에게 급한 연락은? — 요청은 쌓여서 집중 종료 시 전달됩니다. 정말 급하면 재촉(리마인드)을 쓰거나 다른 채널로 연락하세요. "기다려주는 문화"가 이 도구의 핵심입니다.
  • Q. 데이터가 날아가면? — 클라우드 실시간 저장 + 설정의 백업 파일로 이중 보호됩니다.
  • Q. 비용은? — 현재는 무료 운영 중이며, 요금제는 준비 중입니다.
