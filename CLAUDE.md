# OneThing 프로젝트 작업 규칙

## 배포까지 자동 진행 (사용자 지시: 2026-06-25)

이 프로젝트(`C:\AI\OneThing`)에서 **앱 코드(`index.html` 등)를 수정·추가하는 작업을 완료하면**,
사용자의 별도 "배포" 지시를 기다리지 말고 **commit 후 `git push origin main`(배포)까지 이어서** 진행한다.

- 푸시하면 GitHub Pages가 1~2분 뒤 라이브( https://nadoodugui-spec.github.io/onething/ )에 반영됨. 다른 기기는 Ctrl+F5.
- 응답에 **배포 완료(푸시 결과)** 를 함께 보고한다.
- 예외:
  - 설계/계획 문서(`docs/...`)만 바꾼 경우처럼 **앱 동작에 영향이 없으면** 로컬 커밋으로 충분(푸시는 선택).
  - **데이터 영구 삭제 등 되돌리기 어렵거나 위험한 작업**은 푸시 전에 한 번 확인한다.
